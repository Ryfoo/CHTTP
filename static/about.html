About

Scratchy HTTPa server, written by hand

Scratchy is a small HTTP/1.1 server written in C without a framework — just sockets, threads, and a hand-rolled parser. Every request is handed to its own worker thread, so requests don't queue behind one another.

The router is a flat table of (method, path, handler) triples. Anything that doesn't match falls through to a static-file responder for /static/* and /assets/*, then to a 404 page.
